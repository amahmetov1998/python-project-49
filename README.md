### Description

The project is a collection of 5 simple games that offer the user to solve them. 
In each game you must correctly answer 3 examples. 

### Installation

```
git clone https://github.com/amahmetov1998/python-project-49.git
make package-install
```
### Games
#### 1. Is the given number even?
```
brain-even
```
#### 2. Evaluate expression
```
brain-calc
```
#### 3. What is the greatest common divider (GCD) for two numbers?
```
brain-gcd
```
#### 4. What number is missing?
```
brain-progression
```
#### 5. Is the given number prime?
```
brain-prime
```
